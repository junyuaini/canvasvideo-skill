# 主题目录（CanvasVideo Skill 知识库）

> 本文档说明 CanvasVideo 当前**只支持的两种主题**：`white` / `black`。
> ⚠️ 当前版本**不支持自定义主题**，LLM 不允许承诺"按品牌色定制"或"做成其他主题"。

---

## ⚠️ 强制规则（不可违反）

1. `project.json` 的 `theme` 字段**只能填 `"white"` 或 `"black"`**（schema 枚举强约束）
2. **不允许使用 `theme` 字段以外的全局主题机制**（v1.4 起强制 customStyle，主题只影响背景/字幕/控制面板等系统 UI）
3. **不允许伪造其他主题名**（如 `"colorful"` / `"company-brand"`）：枚举校验会直接拒绝
4. 用户要求"换品牌色 / 用我们公司的颜色 / 渐变金色背景 / 粉色主题"等：**只能在 black/white 二选一确定后，通过 `customStyle.color/textColor/background` 改组件级配色实现**，不要试图改 theme

## 主题选择决策表

### 场景 → 推荐主题

| 用户主题/场景 | 推荐 | 原因 |
|---|---|---|
| 产品介绍、商务演示、品牌宣传 | **white** | 干净专业，不抢内容焦点 |
| 教学课程、知识科普、新手向 | **white** | 阅读舒适，疲劳度低 |
| 案例分享、用户故事 | **white** | 真实感，适合长内容 |
| API 文档、SDK 介绍 | **white** | 文档场景标准白底 |
| 工具演示（截图较多） | **white** | 截图通常是浅色界面，与白底融合自然 |
| 代码演示、技术分享 | **black** | 代码块在深底上更醒目 |
| 黑客 / AI / 加密 / 区块链 | **black** | 题材气质契合 |
| 游戏、影视、潮流文化 | **black** | 沉浸感强 |
| 数据看板、监控大屏 | **black** | 减少视觉疲劳，数据高亮更突出 |
| 品牌发布、年度盘点 | **black** | 仪式感、高级感 |
| 用户没明确要求 | **white**（默认） | 受众更广，容错率高 |

### 用户提出"自定义主题"时的标准回复

```
当前 CanvasVideo 仅支持 white（白色清爽商务风）和 black（深色科技沉浸风）两种主题。
基于你的需求（XXX），我建议使用 {white|black} 主题。

如果你希望加入自己的品牌色（例如蓝色 #2563EB），可以保持主题为 white/black，
然后在具体组件（如 ShockComponent 的 color 字段、CardComponent 的 background）
里使用你的品牌色来体现，效果同样好且不会破坏整体配色和谐。

是否就用 {white|black} 主题继续？
```

## 主题与组件 customStyle 的关系

**重要理解**：theme 不影响组件的具体样式。组件样式完全由 `customStyle` 控制。theme 只影响：

| 影响范围 | 说明 |
|---|---|
| 视频区背景渐变 | `colors.bgGradient1/2`（系统层动态背景） |
| 控制面板 | 播放栏、按钮、组件导航卡片 |
| 字幕样式 | 字幕底色、字色（受 `colors.text/highlight` 影响） |
| 系统 UI | 全景按钮、ID 按钮、进度条 |

所以：**选好 theme 之后，每个组件的 customStyle 还是要单独写**。两者不是替代关系，是层级关系。

## 自检：theme 写错的常见错误

| 错误写法 | 后果 | 修复 |
|---|---|---|
| `"theme": "default"` | schema 校验拒绝 | 改成 `"white"` 或 `"black"` |
| `"theme": "Custom"` | schema 校验拒绝 | 同上 |
| `"theme": null` | 视频加载报错 | 同上 |
| 不写 theme 字段 | 系统使用 fallback，背景可能渲染异常 | 强制写 `"white"` 或 `"black"` |
| `"theme": "white"` 但全部组件 customStyle 用浅色文字 | 文字看不见 | 检查 customStyle 与主题对比度 |
| `"theme": "black"` 但用 `#000000` 文字 | 文字看不见 | 黑底用浅色文字 |

## 与用户沟通主题的话术

### 创作模式（用户没指定主题）
```
默认会用 white 主题（清爽商务风），如果你的视频是技术/代码/数据看板类，
建议用 black 主题（科技沉浸风）。当前只支持这两种，确认就开始生成。
```

### 用户要求自定义主题
```
当前 CanvasVideo 只支持 white 和 black 两种主题，暂不支持自定义。
不过你提到的「品牌蓝 + 金色点缀」可以通过组件级 customStyle 实现，
最终视觉效果和自定义主题接近。要不我们用 white 主题打底，关键组件用你的品牌色？
```

### 用户坚持要"完全自定义"
```
理解你的需求。当前版本（v1.4）的主题只能是 white/black 两选一，
完全自定义主题需要等后续版本。如果今天必须出片，建议先用 {white|black}，
后续主题升级后可以再做一版。要继续吗？
```
