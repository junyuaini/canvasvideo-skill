# 视频骨架设计（口播模式）：{主题}

> 口播模式：用户提供音频 + SRT 字幕，AI 负责画面设计。
>
> ⚠️ **核心原则**：字幕是系统自动显示的前提条件，画面是字幕的**视觉翻译**，不是重复。画面设计在区域设计阶段完成。
>
> ⚠️ **关键**：本文件生成后，必须执行 `scripts/generate-skeleton.js` 自动生成 `skeleton.json`，严禁手动编写 JSON。

---

## 项目配置（JSON）

> 填写下方 JSON 代码块，脚本会自动读取并生成 skeleton.json。

```json
{
  "name": "",
  "description": "",
  "theme": "black",
  "duration": 30,
  "viewport": {"width": 780, "height": 585},
  "canvas": {"width": 3300, "height": 700},
  "audio": {"path": "./assets/voice.mp3"},
  "style": "tech",
  "emotion_curve_template": "起伏型",
  "subtitle_count": 8
}
```

**字段说明**：

| 字段 | 必填 | 说明 |
|------|------|------|
| name | ✅ | 视频名称 |
| description | ❌ | 视频描述 |
| theme | ✅ | 背景主题：`white` / `black` |
| duration | ✅ | 总时长（秒），即音频时长 |
| viewport | ✅ | 视口尺寸，默认 `{"width":780,"height":585}` |
| canvas | ✅ | 画布尺寸，**宽度根据区域数自动计算** |
| audio | ✅ | 口播音频配置，`{"path": "音频文件路径"}` |
| style | ✅ | 视觉风格：`warm` / `tech` / `business` / `art` |
| emotion_curve_template | ✅ | 情绪曲线类型：`起伏型` / `渐进型` / `悬念型` / `对比型` / `波浪型` |
| subtitle_count | ✅ | SRT字幕总条数 |

## 区域列表

> 每个区域一行，脚本会自动提取生成 `regions` 数组。

| 名称 | 类型 | 时间段 | 时长(秒) | 包含字幕 | 核心信息 | 情绪 | 位置 | x | y |
|------|------|--------|---------|---------|---------|------|------|---|---|
| P1 | Hook | 0-5s | 5 | 1-2 | ... | 高 | 开头 | 120 | 50 |
| P2 | Point | 5-12s | 7 | 3-5 | ... | 中 | 前1/3段 | 900 | 50 |
| ... | ... | ... | ... | ... | ... | ... | ... | ... | ... |

**字段说明**：

| 字段 | 必填 | 说明 |
|------|------|------|
| 名称 | ✅ | 区域名称：`P{n}` 格式 |
| 类型 | ✅ | 区域类型：Hook/Point/Data/Story/Step/Quote/CTA/Contrast/List/Timeline/Q&A/Scene/Emotion/Summary |
| 时间段 | ✅ | 如 `0-5s`，仅作参考，脚本以时长为准 |
| 时长 | ✅ | 区域持续时间（秒），用于计算时间轴 |
| 包含字幕 | ✅ | 该区域包含的字幕序号范围，如 `1-2` |
| 核心信息 | ❌ | 该区域的核心内容（供用户阅读） |
| 情绪 | ❌ | 该区域的情绪基调：低/中/高（供用户阅读） |
| 位置 | ❌ | 该区域在情绪曲线上的位置：开头/前1/3段/中段/后1/3段/结尾（供用户阅读） |
| x | ✅ | 区域在画布上的 x 坐标 |
| y | ✅ | 区域在画布上的 y 坐标 |

> 注："核心信息"、"情绪"和"位置"列仅用于用户阅读，脚本生成 JSON 时会自动跳过。

**布局坐标规则**：

- 单列布局：y = 50（统一高度）
- 双行布局：第一行 y = 50，第二行 y = 635
- 三行布局：y = 50 / 635 / 1220
- 相邻区域 x 间隔：780（viewport.width）
- 首区域 x 偏移：120
