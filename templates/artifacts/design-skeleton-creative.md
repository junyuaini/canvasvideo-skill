# 视频骨架设计（创作模式）：{主题}

> 创作模式：用户给定主题或文章，AI 自由创作内容、时间轴和画面。
>
> ⚠️ **关键**：本文件生成后，必须执行 `scripts/generate-skeleton.js` 自动生成 `skeleton.json`，严禁手动编写 JSON。

---

## 项目配置（JSON）

> 填写下方 JSON 代码块，脚本会自动读取并生成 skeleton.json。

```json
{
  "name": "",
  "description": "",
  "theme": "white",
  "duration": 15,
  "viewport": {"width": 780, "height": 585},
  "canvas": {"width": 2460, "height": 700},
  "bgm": "tech-pulse"
}
```

**字段说明**：

| 字段 | 必填 | 说明 |
|------|------|------|
| name | ✅ | 视频名称 |
| description | ❌ | 视频描述 |
| theme | ✅ | 背景主题：`white` / `black` |
| duration | ✅ | 总时长（秒） |
| viewport | ✅ | 视口尺寸，默认 `{"width":780,"height":585}` |
| bgm | ✅ | BGM 风格：`tech-pulse` / `warm-cafe` / `uplifting` / `corporate` / `light-pop` / `cinematic` |

**BGM 匹配规则**：

| 视频主题/关键词 | 选用 BGM |
|---|---|
| AI / 编程 / 算法 / 数据 / 科技 | `tech-pulse` |
| 咖啡 / 餐饮 / 生活方式 / 温馨 | `warm-cafe` |
| 创业 / 成长 / 励志 / 教育 | `uplifting` |
| 企业 / B2B / 演示 / 商务 | `corporate` |
| 教程 / 科普 / Vlog / 轻松 | `light-pop` |
| 旅行 / 纪录片 / 大场面 | `cinematic` |
| 不明确 | `corporate`（默认） |

## 区域列表

> 每个区域一行，脚本会自动提取生成 `regions` 数组。

| 名称 | 类型 | 时间段 | 时长(秒) | 核心信息 | 情绪 | 位置 | x | y |
|------|------|--------|---------|---------|------|------|---|---|
| P1 | Hook | 0-3s | 3 | AI时代如何学习 | 高 | 开头 | 120 | 50 |
| P2 | Point | 3-8s | 5 | 三个核心方法 | 中 | 中间 | 900 | 50 |
| ... | ... | ... | ... | ... | ... | ... | ... | ... |

**字段说明**：

| 字段 | 必填 | 说明 |
|------|------|------|
| 名称 | ✅ | 区域名称：`P{n}` 格式 |
| 类型 | ✅ | 区域类型：Hook/Point/Data/Story/Step/Quote/CTA/Contrast/List/Timeline/Q&A/Scene/Emotion/Summary |
| 时间段 | ✅ | 如 `0-3s`，仅作参考，脚本以时长为准 |
| 时长 | ✅ | 区域持续时间（秒），用于计算时间轴 |
| 核心信息 | ❌ | 该区域的核心内容（供用户阅读） |
| 情绪 | ❌ | 该区域的情绪基调：低/中/高（供用户阅读） |
| 位置 | ❌ | 该区域在情绪曲线上的位置：开头/前1/3段/中段/后1/3段/结尾（供用户阅读） |
| x | ✅ | 区域在画布上的 x 坐标 |
| y | ✅ | 区域在画布上的 y 坐标 |

> 注："核心信息"、"情绪"和"位置"列仅用于用户阅读，脚本生成 JSON 时会自动跳过。
> 
> **注意**：区域位置将由前端自动计算（按网格排列），无需手动配置 x/y 坐标。
