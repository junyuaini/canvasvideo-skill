# 区域设计：{区域ID} — {主题}

## 内容规划

### 文案
- 主标题："..."（≤ 15 字）
- 副标题："..."（≤ 30 字）
- 正文："..."（≤ 100 字）
- 标签："..."（≤ 8 字）

### 信息层级
- 层级 1：主标题（最大字号、最醒目颜色）
- 层级 2：副标题/核心数据（中等字号）
- 层级 3：标签、说明文字（小字号）
- 层级 4：图标、背景元素（不干扰阅读）

## 布局设计

**布局模式**：模式一（纯动态）/ 模式二（纯固定）/ 模式三（混合）

```
┌─────────────────────────────┐
│                             │
│     【主标题】               │  ← TitleComponent
│                             │
│  ┌─────────────────────┐   │
│  │    【核心内容】      │   │  ← ShockComponent / CardComponent
│  └─────────────────────┘   │
│                             │
│       [标签]                │  ← BadgeComponent
│                             │
└─────────────────────────────┘
```

## 组件清单

| ID | 类型 | 内容 | 位置 (x, y, w, h) |
|----|------|------|-------------------|

**组件数量限制**：
- TitleComponent：每区域 1 个
- TextComponent：每区域 ≤ 2 个
- ShockComponent：每区域 ≤ 1 个
- BadgeComponent：每区域 ≤ 2 个
- ImageComponent：每区域 ≤ 1 个
- CardComponent：每区域 ≤ 2 个
- GraphicComponent：每区域 ≤ 1 个
- QuoteComponent：每区域 ≤ 1 个
- **总计 ≤ 5 个**
