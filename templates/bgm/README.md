# BGM 目录

内置 6 首免版权 BGM，供创作模式自动匹配。

## 文件清单

| 风格 | 文件名 |
|------|--------|
| tech | `tech-pulse.mp3` |
| warm | `warm-cafe.mp3` |
| uplifting | `uplifting.mp3` |
| corporate | `corporate.mp3` |
| light-pop | `light-pop.mp3` |
| cinematic | `cinematic.mp3` |

## 用法

BGM 在初始化时选定，风格匹配写入 project.json。

## 替换

如需替换，保持同名文件覆盖即可。
