# 布局规则

> 画布尺寸、区域排布、组件坐标、尺寸约束。

---

## R1 viewport 与 regions

| 项 | 规则 | 值 |
|---|---|---|
| 默认 viewport | 用户未指定时用 | 780×585 |
| 区域排布 | 每行4个，超4换行 | — |
| 行间距 | 固定 | 55px |
| 列间距 | 固定 | 30px |
| 区域 margin | 固定 | 20px |
| canvas 宽度 | 最右区域x + viewport.w + 60 | — |
| canvas 高度 | 最下区域y + viewport.h + 60 | — |
| canvas 最小高度 | 单区域时 | viewport.h + 60 |
| canvas 最小宽度 | 单区域时 | viewport.w + 60 |

**区域布局坐标**（每行4列）：

| 区域索引 | 行/列 | x | y |
|---|---|---|---|
| P1 | 行1·列1 | 120 | 50 |
| P2 | 行1·列2 | 900 | 50 |
| P3 | 行1·列3 | 1680 | 50 |
| P4 | 行1·列4 | 2460 | 50 |
| P5 | 行2·列1 | 120 | 635 |

**canvas 尺寸对照**：

| 区域数 | 行数 | 推荐 canvas |
|---|---|---|
| 1-4 | 1 行 | `width: 3300, height: 700` |
| 5-8 | 2 行 | `width: 3300, height: 1285` |
| 9-12 | 3 行 | `width: 3300, height: 1870` |
| 13-16 | 4 行 | `width: 3300, height: 2455` |

**严禁**：
- ❌ 每行5-6个区域
- ❌ 行间距不固定
- ❌ 区域 margin 不一致

## R2 组件 Y 坐标

| 项 | 规则 |
|----|------|
| Y 连续递增 | sorted[i].y ≥ sorted[i-1].y + h + 10 |
| 强制间距 | 相邻组件间距 ≥ 10px |

**错误示例**：
```jsonc
// ❌ 后组件 y < 前组件 bottom
{ "y": 0, "h": 595 },
{ "y": 0, "h": 100 }
```

**正确示例**：
```jsonc
// ✅ Y 递增
{ "y": 0, "h": 100 },
{ "y": 110, "h": 200 }
```

## R3 尺寸约束

| 项 | 规则 |
|----|------|
| 组件 w | ≤ viewport.width - 40 |
| 组件 h 总和 | + 间距 ≤ viewport.height - 20 |
| 强调类组件居中 | Shock/Badge/CTA 单独出现时水平居中 |
| position.w / position.h | 必须**显式填写** |

---

## R4 对比度

| 项 | 规则 |
|----|------|
| 文字对比度 | 符合 WCAG AA（≥ 4.5:1） |
| 背景色 | 纯白 #FFFFFF 或纯黑 #0F172A |
| 文字色 | 纯白 #FFFFFF 或深灰 #111827 |
