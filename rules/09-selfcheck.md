# 自检规则

> 视频生成后的自检检查表与评分标准。

---

## R1 自检层级

| 层级 | 检查项 | 通过条件 |
|------|--------|---------|
| L0 | 文件完整性 | output.zip 存在且可解压 |
| L1 | JSON 结构 | project.json 合法、字段完整 |
| L2 | 组件规范 | 组件数量≤5、ID唯一、时间无重叠、Y递增 |
| L3 | 视觉丰富度 | 6条门槛 ≥4分 |
| L4 | 音频同步 | 配音模式：字幕覆盖音频±0.5s；BGM模式：无subtitles |

---

## R2 评分标准

| 分数 | 状态 |
|------|------|
| 0 | 未通过任何层级 |
| 1 | 通过 L0 |
| 2 | 通过 L0+L1 |
| 3 | 通过 L0+L1+L2 |
| 4 | 通过 L0+L1+L2+L3 |
| 5 | 通过全部层级 |

**目标**：≥ 4 分（及格线）

---

## R3 自检流程

1. 运行 `scripts/validate.js` 自动检查 L0~L2
2. 对照 `rules/05-richness.md` 检查 L3
3. 对照 `rules/02-mode.md` §R4 检查 L4
4. 汇总评分，低于 4 分需修复
