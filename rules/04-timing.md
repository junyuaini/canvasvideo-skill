# 节奏规则

> 区域时长、时间轴节奏、settings 参数。

---

## R1 区域时长门槛

| 模式 | 最大区域时长 | 最小区域数（按总时长） |
|------|------------|----------------------|
| 创作模式 | ≤ 5 秒/区 | 30s≥6 / 60s≥12 / 120s≥24 / 180s≥36 |
| 口播模式 | ≤ 8 秒/区 | 30s≥4 / 60s≥8 / 120s≥15 / 180s≥23 |

**严禁**：
- ❌ 创作模式区域时长 > 5s
- ❌ 口播模式区域时长 > 8s

---

## R2 组件时间轴节奏

| 项 | 创作模式 | 口播模式 |
|----|---------|---------|
| 末组件停留 | ≤ 1s | ≤ 2s |
| 相邻组件间隔 | ≤ 1s | ≤ 3s |
| 关键信息停留 | ≥ 2s | ≥ 2s |

**全局绝对时间**：组件 start/end 基于骨架设计中的区域时间范围，不是相对时间。

**严禁**：
- ❌ 组件 start < 区域 start
- ❌ 组件 end > 区域 end
- ❌ 组件时间重叠冲突

---

## R3 settings 参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `preFullViewDuration` | 0.4 | 区域进入视口前的停留时间 |
| `postFullViewDuration` | 0.4 | 区域离开视口后的停留时间 |
| `contentZoomRatio` | 0.9 | 内容缩放比例 |

**严禁**：
- ❌ 使用未定义的 settings 字段
