# 节奏规则

> 区域时长、时间轴节奏、settings 参数。

---

## R1 区域时长门槛

| 模式 | 最大区域时长 | 最小区域数（按总时长） |
|------|------------|----------------------|
| 创作模式 | ≤ 5 秒/区 | 30s≥6 / 60s≥12 / 120s≥24 / 180s≥36 |
| 口播模式 | ≤ 8 秒/区 | 30s≥4 / 60s≥8 / 120s≥15 / 180s≥23 |

**严禁**：
- ❌ 创作模式区域时长 > 5s
- ❌ 口播模式区域时长 > 8s

---

## R4 口播模式语义分段原则（核心）

**原则**：按语义自然分段，不是按句数硬切。

**判断语义边界的信号**：
- 话题转换（从"问题"转到"解决方案"）
- 逻辑转折（"但是"、"然而"、"不过"）
- 步骤递进（"第一步"、"第二步"、"最后"）
- 情绪变化（从"痛点"转到"希望"）
- 时间跳跃（"过去"→"现在"→"未来"）

**分段约束**：
- 每区域时长 ≤ 8 秒
- 每区域至少包含 1 条字幕
- 语义边界处切分（不在句子中间切）
- 一个完整意思 = 一个区域

---

## R2 组件时间轴节奏

| 项 | 创作模式 | 口播模式 |
|----|---------|---------|
| 末组件停留 | ≤ 1s | ≤ 2s |
| 相邻组件间隔 | ≤ 1s | ≤ 3s |
| 关键信息停留 | ≥ 2s | ≥ 2s |

**全局绝对时间**：组件 start/end 基于骨架设计中的区域时间范围，不是相对时间。

**严禁**：
- ❌ 组件 start < 区域 start
- ❌ 组件 end > 区域 end
- ❌ 组件时间重叠冲突

---

## R3 settings 参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `preFullViewDuration` | 0.4 | 区域进入视口前的停留时间 |
| `postFullViewDuration` | 0.4 | 区域离开视口后的停留时间 |
| `contentZoomRatio` | 0.9 | 内容缩放比例 |

**严禁**：
- ❌ 使用未定义的 settings 字段
