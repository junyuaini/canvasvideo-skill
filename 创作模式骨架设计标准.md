# 创作模式骨架设计标准

## 模式定义
用户给定主题或文章，AI 自由创作内容、时间轴和画面。一切由 AI 设计和生成。

---

## 输入

| 字段 | 必填 | 说明 |
|------|------|------|
| 主题/文章内容 | 是 | 用户提供的原始素材 |
| 总时长 | 否 | 默认自动推断，用户可指定 |
| 风格 | 否 | warm/tech/business/art，默认 warm。产品自定义的4种风格主题 |
| 背景 | 否 | black/white，默认 black |
| 字幕 | 否 | true/false，默认 false |
| 音乐 | 否 | auto/none，默认 auto |

---

## 输出

区域列表，每个区域包含：
- 序号
- 类型
- 起止时间
- 时长
- 核心信息（一句话概括）
- 情绪强度（低/中/高）
- 区域位置（开头/前1/3/中段/后1/3/结尾）

---

## 前置概念（设计前需了解）

### 1. 区域类型

| 类型 | 功能 | 适用场景 | 建议时长 |
|------|------|---------|---------|
| Hook | 开头抓眼 | 任何视频开头 | 2-3 秒 |
| Point | 核心观点 | 观点文、结论段 | 3-5 秒 |
| Data | 数字统计 | 报告、论证 | 4-6 秒 |
| Story | 场景案例 | 故事文、情感文 | 5-8 秒 |
| Step | 方法流程 | 教程文、指南 | 5-10 秒 |
| Quote | 引用证言 | 增强可信度 | 3-5 秒 |
| CTA | 结尾引导 | 任何视频结尾 | 3-5 秒 |
| Contrast | 对比反差 | 论证、说服 | 4-6 秒 |
| List | 列举清单 | 盘点、推荐 | 5-8 秒 |
| Timeline | 时间线 | 历史、回顾 | 5-10 秒 |
| Q&A | 问答设问 | 科普、解惑 | 3-5 秒 |
| Scene | 场景还原 | 故事、沉浸 | 5-8 秒 |
| Emotion | 情绪渲染 | 情感、共鸣 | 3-5 秒 |
| Summary | 总结归纳 | 中段或结尾 | 3-5 秒 |

### 2. 风格定义（产品自定义）

| 风格 | 对外名称 | 适用内容 | 视觉特征 |
|------|---------|---------|---------|
| warm | 温情风 | 情感、故事、生活 | 暖色调、柔和、多用Story/Emotion/Scene |
| tech | 科技风 | 数据、产品、未来 | 冷色调、几何、多用Data/Step/Timeline |
| business | 商务风 | 商业、职场、专业 | 深蓝/灰色、稳重、多用Point/Data/Contrast |
| art | 艺术风 | 创意、文化、美学 | 大胆配色、设计感、多用Scene/Emotion/Quote |

### 3. 情绪强度

| 强度 | 含义 | 画面特征 |
|-----|------|---------|
| 低 | 平静、克制、留白 | 元素少、节奏慢、颜色淡 |
| 中 | 正常叙述、信息传递 | 元素适中、节奏平稳 |
| 高 | 冲击、高潮、强调 | 元素多、节奏快、颜色饱和 |

---

## 设计步骤

### 步骤1：确定风格和背景

| 输入 | 说明 | 默认值 |
|------|------|--------|
| 风格 | warm/tech/business/art | warm |
| 背景 | black/white | black |

**风格选择依据**：
- 内容偏向情感、故事、生活 → warm
- 内容偏向数据、产品、技术 → tech
- 内容偏向商业、职场、专业 → business
- 内容偏向创意、文化、美学 → art

**背景选择依据**：
- 用户未指定时，默认 black
- 商务、教学、科普 → white
- 代码、AI、技术、数据 → black

---

### 步骤2：推断总时长和区域数

| 内容长度 | 建议总时长 | 区域数 | 情绪曲线建议 |
|---------|-----------|--------|------------|
| 一句话主题 | 10-15 秒 | 3 个 | 起伏型 |
| 一段短文（100字内） | 15-30 秒 | 3-4 个 | 起伏型 |
| 一篇文章（500字内） | 30-60 秒 | 4-5 个 | 起伏型 |
| 长文（1000字内） | 60-90 秒 | 5-7 个 | 波浪型 |
| 深度长文（2000字内） | 90-120 秒 | 7-9 个 | 波浪型 |
| 专题报告（3000字+） | 120-180 秒 | 9-12 个 | 波浪型 |
| 纪录片/课程章节 | 180-300 秒 | 12-15 个 | 波浪型+分段高潮 |
| 系列课程/长纪录片 | 300秒+ | 15-20 个 | 波浪型+章节结构 |

用户指定时长时，按指定值拆分；未指定时，按内容长度自动推断。

---

### 步骤3：选择情绪曲线模板

| 模板 | 曲线 | 适合场景 |
|------|------|---------|
| 起伏型 | 中→高→中→高→中 | 观点文、故事文、大多数短视频 |
| 渐进型 | 低→中→高→更高→中 | 发布会、演讲、励志内容 |
| 悬念型 | 高→低→中→高→低 | 悬疑、反转、故事类 |
| 对比型 | 中→高→低→高→中 | 对比论证、正反分析 |
| 波浪型 | 中→高→中→高→低→中→高→中 | 60秒以上长视频、纪录片 |

**选择原则**：
- 60秒以下：优先起伏型
- 60-90秒：优先波浪型
- 90秒以上：波浪型+章节结构

---

### 步骤4：拆分信息单元并确定区域类型

**3.1 拆分原则**

1. 一段一区域：每个自然段或信息单元对应一个区域
2. 信息单一：一个区域只讲一个点
3. 时长匹配：每个区域3-10秒
4. 节奏交替：紧张区域和松弛区域交替
5. 类型不重复：相邻区域类型尽量不重复

**3.2 用决策树确定每个区域的类型**

**第一层：内容目的**
```
区域的核心目的是什么？
├─ 吸引注意力 → Hook
├─ 传递信息 → Data/Point/Step/List/Timeline
├─ 建立情感 → Story/Scene/Emotion/Quote
├─ 制造冲突 → Contrast/Q&A
└─ 收束引导 → Summary/CTA/Point
```

**第二层：情绪曲线位置**
```
该区域在情绪曲线上的位置？
├─ 开头（0-3秒）→ 必须为Hook，情绪不能低
├─ 前1/3段 → 优先Point/Data/List/Scene，情绪中→高
├─ 中段 → 呼吸点用Contrast/Summary，记忆点用Story/Emotion
├─ 后1/3段 → 冲击用Emotion/Contrast，论证用Story/Data
└─ 结尾 → 优先CTA/Summary/Quote，情绪不能高
```

**第三层：风格匹配**
```
整体风格是什么？
├─ warm → 优先Story/Emotion/Scene
├─ tech → 优先Data/Step/Timeline
├─ business → 优先Point/Data/Contrast
└─ art → 优先Scene/Emotion/Quote
```

**三层递进，不是三选一**。每层都要考虑，共同决定最终类型。

**3.3 边界处理**

- 类型重复但内容不同 → 允许，中间插入过渡区域
- 类型重复且内容相似 → 更换类型或合并
- 时长超过上限 → 拆分为两个区域
- 时长低于下限 → 与相邻区域合并

---

### 步骤5：分配情绪强度

根据情绪曲线模板和区域位置，分配每个区域的情绪强度：

| 区域位置 | 起伏型 | 渐进型 | 悬念型 | 对比型 | 波浪型 |
|---------|--------|--------|--------|--------|--------|
| 开头 | 中 | 低 | 高 | 中 | 中 |
| 前1/3 | 高 | 中 | 低 | 高 | 高 |
| 中段 | 中 | 高 | 中 | 低 | 中 |
| 后1/3 | 高 | 更高 | 高 | 高 | 高 |
| 结尾 | 中 | 中 | 低 | 中 | 中 |

**情绪曲线设计原则**：
1. 避免平线
2. 开头有钩子（前3秒不能低）
3. 结尾有余韵（最后不能高）
4. 高潮在黄金点（60秒视频在20-40秒）
5. 相邻不重复
6. 低谷有价值（用于过渡、反思）

---

### 步骤6：长视频章节划分（90秒以上）

**5.1 章节定义**

- 每60-90秒为一个章节
- 每个章节有自己的情绪小循环
- 章节间用过渡区域衔接
- 章节结尾情绪不能高（为下一章蓄力）

**5.2 五种章节微循环**

| 微循环 | 曲线 | 适用 |
|--------|------|------|
| 标准起伏型 | 中→高→中 | 大多数章节 |
| 渐进攀升型 | 低→中→高 | 章节初期铺垫 |
| 悬念蓄力型 | 中→低→中 | 章节结尾衔接 |
| 双高潮型 | 中→高→中→高→低 | 信息密集章节 |
| 波浪递进型 | 低→中→高→中→高 | 深度内容章节 |

**5.3 章节间衔接**

| 模式 | 章节1结尾 | 过渡 | 章节2开头 | 适用 |
|------|----------|------|----------|------|
| 平滑过渡 | 情绪降至中/低 | Summary | Hook | 通用 |
| 悬念过渡 | Contrast制造悬念 | Q&A | Scene | 故事类 |
| 直接切换 | Point强观点 | 无（视觉转场） | Data | Tech风格 |

**5.4 章节数量映射**

| 视频时长 | 章节数 | 每章时长 | 推荐组合 |
|---------|--------|---------|---------|
| 90-120秒 | 2个 | 45-60秒 | A+C |
| 120-180秒 | 3个 | 40-60秒 | A+D+B |
| 180-240秒 | 3-4个 | 50-70秒 | A+D+C+B |
| 240-300秒 | 4-5个 | 50-65秒 | A+D+C+B+A |
| 300秒+ | 5-8个 | 45-70秒 | 交替使用，避免相邻重复 |

---

### 步骤7：输出骨架

**6.1 骨架模板**

```
视频主题：__
总时长：__秒
风格：__
背景：__
情绪曲线模板：__

区域列表：
1. [类型] __，__秒，核心信息：__，情绪：__，位置：__
2. [类型] __，__秒，核心信息：__，情绪：__，位置：__
...

时间轴：
0.0s - __s  区域1
__s - __s   区域2
...

情绪曲线：
[低/中/高] → [低/中/高] → ...
```

**6.2 自检清单**

- [ ] 区域数量符合时长推断规则
- [ ] 每个区域时长3-10秒
- [ ] 总时长之和等于目标时长
- [ ] 区域类型不重复超过2次
- [ ] 相邻区域类型不同
- [ ] 有Hook或CTA至少其一
- [ ] 情绪曲线有起伏，不是平线
- [ ] 开头情绪不是"低"
- [ ] 结尾情绪不是"高"
- [ ] 每个区域核心信息能用一句话说清
- [ ] 信息密度适中，没有区域塞入多个主题
- [ ] 90秒以上视频有章节划分
- [ ] 类型选择符合决策树（内容目的×情绪位置×风格）
- [ ] 长视频章节内使用了微循环模板
- [ ] 章节间过渡区域设置合理

---

## 示例

### 示例1：60秒"2024年AI行业报告"

```
视频主题：2024年AI行业报告
总时长：60秒
风格：tech
背景：black
情绪曲线模板：起伏型

区域列表：
1. [Hook] 钩子，3秒，核心信息：AI正在改变一切，情绪：高，位置：开头
2. [Data] 市场规模，5秒，核心信息：全球AI市场增长300%，情绪：高，位置：前1/3
3. [List] 三大突破，8秒，核心信息：大模型、多模态、具身智能，情绪：中，位置：前1/3
4. [Story] 应用案例，10秒，核心信息：某医院用AI诊断提速10倍，情绪：中，位置：中段
5. [Contrast] 风险与挑战，8秒，核心信息：伦理问题与数据安全，情绪：低，位置：中段
6. [Timeline] 未来展望，8秒，核心信息：2025-2030发展预测，情绪：中，位置：后1/3
7. [Point] 核心观点，6秒，核心信息：AI是工具，关键在人，情绪：高，位置：后1/3
8. [CTA] 收束，4秒，核心信息：关注我们，拥抱AI时代，情绪：中，位置：结尾

时间轴：
0.0s - 3.0s    区域1 钩子        [高]
3.0s - 8.0s    区域2 市场规模     [高]
8.0s - 16.0s   区域3 三大突破     [中]
16.0s - 26.0s  区域4 应用案例     [中]
26.0s - 34.0s  区域5 风险与挑战   [低]
34.0s - 42.0s  区域6 未来展望     [中]
42.0s - 48.0s  区域7 核心观点     [高]
48.0s - 52.0s  区域8 收束         [中]

情绪曲线：高 → 高 → 中 → 中 → 低 → 中 → 高 → 中
```

### 示例2：180秒长视频章节结构

```
视频主题：人工智能发展史
总时长：180秒
风格：tech
背景：black
情绪曲线模板：波浪型

章节1（0-60秒）：引入+起源，微循环：标准起伏型
0.0s - 5.0s    区域1 Hook       情绪：中  "1956年，一个夏天的会议"
5.0s - 15.0s   区域2 Scene      情绪：高  "达特茅斯：十个人，两个月，一个梦想"
15.0s - 25.0s  区域3 Timeline   情绪：中  "从符号主义到连接主义"
25.0s - 35.0s  区域4 Story      情绪：高  "第一次AI寒冬"
35.0s - 45.0s  区域5 Data       情绪：中  "80年代专家系统的辉煌与崩溃"
45.0s - 55.0s  区域6 Summary    情绪：中  "早期AI教会了我们什么"
55.0s - 60.0s  【过渡】Scene    情绪：低  "画面从黑白转入彩色"

章节2（60-120秒）：突破+应用，微循环：双高潮型
60.0s - 68.0s  区域7 Point      情绪：中  "2012年，一切开始改变"
68.0s - 78.0s  区域8 Data       情绪：高  "ImageNet：深度学习的黎明"
78.0s - 88.0s  区域9 Story      情绪：中  "从实验室到产业"
88.0s - 98.0s  区域10 Emotion   情绪：高  "AlphaGo：标志性时刻"
98.0s - 108.0s 区域11 List      情绪：中  "AI三大应用领域"
108.0s - 115.0s区域12 Contrast  情绪：高  "赋能还是替代？"
115.0s - 120.0s【过渡】Q&A     情绪：低  "AI会取代人类吗？"

章节3（120-180秒）：现状+未来，微循环：渐进攀升型
120.0s - 128.0s区域13 Contrast  情绪：低  "机遇与挑战并存"
128.0s - 138.0s区域14 Data      情绪：中  "2024年：大模型爆发"
138.0s - 148.0s区域15 Timeline  情绪：中  "未来五年发展预测"
148.0s - 158.0s区域16 Point     情绪：高  "关键是人怎么用它"
158.0s - 168.0s区域17 Emotion   情绪：高  "未来属于善用AI的人"
168.0s - 175.0s区域18 Summary   情绪：中  "历史不会重复，但会押韵"
175.0s - 180.0s区域19 CTA       情绪：中  "加入我们，探索AI的下一个十年"

情绪曲线（章节级）：
中→高→中→高→中→中→【过渡】→中→高→中→高→中→高→【过渡】→低→中→中→高→高→中→中
```
